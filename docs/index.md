# Scam Shield

Advanced Threat Detection Platform combining Scam Shield and Scam Shield AI.

## Overview

Scam Shield is an advanced threat detection platform designed to identify and mitigate online scams, phishing attacks, and other cyber threats in real-time.

## Features

- **Threat Scanner**: Analyze text, URLs, or emails for scam indicators
- **AI Integration**: Machine learning-powered detection of phishing, impersonation, and malware signals
- **Dashboard**: Scan history, threat levels, and alerts
- **Reporting**: Generate reports on detected threats

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
